Video Downloader Bot (VK & Rutube)
Простой и эффективный Telegram-бот для скачивания видео из социальных сетей ВКонтакте и Rutube. Просто отправь ссылку — получи видеофайл.
## ✨ Особенности
🚀 Поддержка ВК Видео: скачивание клипов и обычных видео.
📺 Поддержка Rutube: загрузка видео в высоком качестве.
⚡ Быстрая отправка: бот оптимизирован для работы с большими файлами.
🛠 Простая установка: минимум зависимостей и понятная настройка.
🛠 Технологии
Язык: Python
Библиотека бота: aiogram (или telebot)
Загрузчик: yt-dlp (универсальный инструмент для работы с видео)
